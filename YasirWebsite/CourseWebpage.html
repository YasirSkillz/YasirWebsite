Course Description
Intro To Programming (Java) I learned to efficiently use an IDE for Java development, covering editing, compiling, running, and debugging. The course includes defining variables, using operators, type conversions, basic I/O operations, branching, looping, file handling, creating Java classes with inheritance, and working with arrays. It ends with creating a basic GUI program with event handling.
Composition I In this course, I mastered tailoring my writing to suit specific audiences and purposes, generated ideas through effective pre-writing activities, crafted focused essays with clear, supportable thesis statements, developed coherent paragraphs, utilized MLA or APA format and documentation, and proficiently identified and corrected grammar and mechanics errors.
HTML5 Programming I learned how to create HTML5 documents, adding content through elements, connecting pages with hyperlinks, capturing user input via forms, creating CSS stylesheets for layout, and implementing basic program logic using JavaScript.
World Literature This course helped me analyze texts and taught me about translations. We practiced our writing skills.
Focus This class is mainly my recreational class. There aren't many educational purposes for this class and it is used as time to get work done for other classes.
